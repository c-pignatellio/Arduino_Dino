## Descrição do projeto
O projeto se resume em um jogo em que o player tem que pular obstáculos que aparecem em sua frente. Há uma pontuação instantânea e uma pontuação máxima, que, durante todo o tempo de execução, guarda a maior pontuação alcançada.  
O hardware é composto por componentes simples: Controlador Arduíno Uno, um botão 6mmx6mmx5mm, um Display OLED 0.96 I2C e alguns jumpers macho-macho e macho-fêmea (protoboard a se decidir).
O código foi baseado em um comum retirado do site oficial do Arduino e aprimorado com nossas preferências, e uma simulação do funcionamento foi feito na plataforma Wokwi.  

Link do projeto de inspiração (posso tirar de for o caso): https://projecthub.arduino.cc/ultramegabombasticfuze/chrome-dino-game-b10afe  
Link do projeto no Wokwi (de autoria própria): https://wokwi.com/projects/467812653743745025
